## 🚀 Quick Start

1. **Clone & Install**
   ```bash
   git clone <repository-url>
   cd resume
   npm install
   ```

2. **Setup Firebase**
   - Create a Firebase project
   - Enable Google Authentication
   - Create a Firestore database
   - Copy your Firebase config

3. **Environment Setup**
   ```bash
   cp .env.example .env
   # Add your Firebase configuration
   ```

4. **Run the App**
   ```bash
   npm start
   ```
